# 第五章 回避

## 回避的概念和意义

### 回避的概念

有关与案件或者案件的当事人有某种特定关系的审判人员、检察人员、侦查人员等不得参加该案件审判、检察、侦查等活动的一项诉讼制度

### 回避的意义

1. 有利于案件得到客观、公正的处理
2. 有利于保持司法在公众心目中的威信

## 回避的适用范围和理由

### 回避的适用范围

审判人员、检察人员、侦查人员、书记员、翻译人员、鉴定人

### 回避的形式

自行回避 申请回避  指令回避

### 回避的理由

1. 是本案的当事人或者是当事人的近亲属的
2. 本人或者他的近亲属与本案有利害关系的
3. 担任过本案的证人、鉴定人、辩护人、诉讼代理人的
4. 与本案当事人有其他关系，可能影响公正处理案件的
5. 接受当事人及其委托的人的请客送礼或者违反规定会见当事人及其委托的人的
6. 其他情形的
