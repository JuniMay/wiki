# 第十三章 审查起诉

## 提起公诉的概念、意义

### 提起公诉的概念

!!! info "概念"


!!! info "起诉制度的分类"

    1. 统一公诉制
    2. 公诉兼自诉

### 公诉制度的方式

起诉法定主义、起诉便宜主义

### 意义

- 为审判做准备
- 保障无罪的人不受刑事追究
- 起诉的内容决定审判的范围
  - 未起诉的事实不得审判
  - 未起诉的人不得审判
  - 审判结论所确认的事和人必须与起诉指控的事实和人相一致

## 监察调查与审查起诉的衔接

### 对监察委移交案件审查起诉的理由

人民检察院对监察机关移送起诉的案件依本法与**监察法**进行审查。

### 退回补充调查和自行补充调查

人民检察院经审查，认为需要补充核实的，应当退回监察机关补充调查，必要时可以自行补充侦查。

### 留置与强制措施的转换及期限

对于监察机关移送起诉的已采取**留置**措施的案件，人民检察院应当对犯罪嫌疑人**先行拘留**，留置措施**自动解除**

人民检察院应当在拘留后的 **十日** 以内作出是否逮捕、取保候审或者监视居住的u而定，在特殊情况下，决定的时间可以延长一日至四日

人民检察院决定采取强制措施的期间不计入审查起诉期限。

## 审查起诉

!!! info "概念"

    指人民检察院对公安机关侦查终结移送起诉的案件，以及自行侦查终结的案件，进行审查，以确定是否符合起诉条件的诉讼活动。

### 审查起诉的权力主体

凡需要提起公诉的案件，**一律** 由人民检察院审查决定。

### 审查起诉的内容

- 犯罪嫌疑人的身份是否清楚 [^1]

[^1]: 不是绝对的，原则上要求清楚，实在不清楚时依照犯罪嫌疑人自报的身份起诉。

- 犯罪事实、情节是否清楚，证据是否确实、充分
- 犯罪性质和罪名的认定是否正确
- 有无遗漏罪行和其他应当追究刑事责任的人
- 是否属于依法不追究刑事责任的情况
- 有无附带民事诉讼
- 侦查活动是否合法

### 审查起诉的程序

　　人民检察院审查案件，应当讯问犯罪嫌疑人，听取辩护人、被害人及其诉讼代理人的意见，并记录在案。辩护人、被害人及其诉讼代理人提出书面意见的，应当附卷。

### 审查起诉的期限

- 人民检察院对于公安机关移送起诉的案件，应当在 **一个月以内** 做出决定，重大、复杂的案件，可以延长十五日；
- 犯罪嫌疑人 **认罪认罚**，符合速裁程序适用条件的，应当在 **十日** 以内做出决定，对可能判处的有期徒刑超过一年的，可以延长至 **十五日**。
- 人民检察院审查起诉的案件，改变管辖的，从改变后的人民检察院收到案件之日起计算审查起诉期限。

### 审查起诉的程序

!!! quote "第一百七十三条"

    人民检察院审查案件，应当讯问犯罪嫌疑人，听取辩护人或者值班律师、被害人及其诉讼代理人的意见，并记录在案。辩护人或者值班律师、被害人及其诉讼代理人提出书面意见的，应当附卷。

    犯罪嫌疑人认罪认罚的，人民检察院应当告知其享有的诉讼权利和认罪认罚的法律规定，听取犯罪嫌疑人、辩护人或者值班律师、被害人及其诉讼代理人对下列事项的意见并记录在案：

    1. 涉嫌的犯罪事实、罪名及适用的法律规定；
    2. 从轻、减轻或者免除处罚等从宽处罚的建议；
    3. 认罪认罚后案件审理适用的程序
    4. 其他需要听取意见的事项

    人民检察院依照前两款规定听取值班律师意见的，应当提前为值班律师了解案件有关情况提供必要的便利。

### 认罪认罚具结书的签署

犯罪嫌疑人自愿认罪，同意量刑建议和程序适用的，应当在辩护人或者值班律师在场的情况下签署认罪认罚具结书 [^2]。

[^2]: 保证性质，单方性质。

犯罪嫌疑人认罪认罚，有下列情形之一的，不需要签署认罪认罚具结书：

- 犯罪嫌疑人是盲、聋、哑人，或者是尚未完全丧失辨认或者控制自己行为能力的精神病人的；
- 未成年犯罪嫌疑人的法定代理人、辩护人对未成年人认罪认罚有异议的；
- 其他不需要签署认罪认罚具结书的情形


### 证据的提供及非法证据的排除

　　人民检察院审查案件，可以要求公安机关提供法庭审判所必需的证据材料；认为可能存在本法第五十四条规定的以非法方法收集证据情形的，可以要求其对证据收集的合法性作出说明。


### 审查后的处理

- 提起公诉
- 不起诉

## 提起公诉

!!! info "概念"

人民检察院认为犯罪嫌疑人的犯罪事实已经查清，证据确实、充分，从而决定将犯罪嫌疑人提交人民法院进行审判的诉讼活动。

### 条件

- 犯罪嫌疑人的犯罪事实以已经查清，证据确实、充分
- 依法应当追究犯罪嫌疑人的刑事责任

### 提起公诉的程序

1. 制作公诉书；
2. 按照审判管辖的规定，向人民法院提起公诉；
3. **并将案卷材料、证据移送人民法院。**
4. 人民法院审判公诉案件，人民检察院应当派员出席法庭支持公诉
5. 犯罪嫌疑人认罪 **认罚** 的，人民检察院应当就主刑、附加刑、是否适用缓刑等提出量刑建议，并随案移送认罪认罚具结书等材料。

## 不起诉

!!! info "概念"

    人民检察院结果审查，认为犯罪嫌疑人的情况不符合起诉条件，从而不将犯罪嫌疑人提交人民法院进行审判的决定。

### 不起诉的种类和条件

- 法定不起诉（绝对不起诉）
  
    犯罪嫌疑人没有犯罪事实，或者由本法第十五条规定的情形之一的，人民检察院应当作出不起诉决定。

- 酌定不起诉[^3]
  
    [^3]: 从“免于起诉”而来
  
    对于犯罪情节轻微，依照刑法规定不需要判处刑罚或者免除刑罚的，人民检察院可以作出不起诉决定。

- 存疑不起诉
  
    对于二次补充侦查的案件，人民检察院仍然认为证据不足，不符合起诉条件的，**应当** 作出不起诉的决定。

[ ] 第一百八十二条
[ ] 附条件不起诉
[ ] （企业）合规不起诉
  
### 不起诉的程序

1. 不起诉决定书的制作及公开宣布
   1. 不起诉应当制作不起诉决定书
   2. 不起诉的决定，应当公开宣布
2. 释放被羁押的被不起诉人

    如果不起诉人在押，应当立即释放

3. 不起诉决定书的送达
   1. 不起诉决定书应当送达被不起诉人和他的所在单位
   2. 对于公安机关移送起诉的案件，人民检察院决定不起诉的，应当将不起诉决定书送达公安机关。
   3. 对于有被害人的案件，决定不起诉的，人民检察院应当将不起诉决定书送达被害人。
4. 公安机关对不起诉决定的复议、复核申请

    公安机关认为不起诉的决定有错误的时候，可以要求复议，如果意见不被接受，可以向上一级人民检察院提请复核。

5. 被不起诉人对酌定不起诉的申诉、人民检察院的复查
   1. 对于人民检察院依照本法第一百七十三条第二款规定作出的不起诉决定，被不起诉人如果不服，可以自收到决定书后七日以内向人民检察院申诉。
   2. 人民检察院应当作出复查决定，通知被不起诉的人，同时抄送公安机关。
6. 被害人对不起诉的案件可以提起自诉
   1. 被害人如果不服，可以自收到决定书后七日以内向上一级人民检察院中诉，请求提起公诉。
   2. 人民检察院应当将复查决定告知被害人。
   3. 对人民检察院维持不起诉决定的，被害人可以向人民法院起诉。
   4. 被害人也可以不经申诉，直接向人民法院起诉。
   5. 人民法院受理案件后，人民检察院应当将有关案件材料移送人民法院。

### 查封、扣押、冻结的财物的处理

1. 人民检察院决定不起诉的案件，应当同时对侦查中查封、扣押、冻结的财物解除查封扣押、冻结。
2. 对被不起诉人需要给子行政处罚、处分或者需要没收其违法所得的，人民检察院应当提出检察意见，移送有关主管机关处理。
3. 有关主管机关应当将处理结果及时通知人民检察院。

### 重大立功或者案件涉及国家重大利益的处理

1. 犯罪嫌疑人资源如实供述涉嫌犯罪的事实，有重大立功或者案件涉及国家重大利益的经最高人民检察院核准，公安机关可以撤销案件，人民检察院可以作出不起诉决定，也可以对涉嫌数罪中的一项或者多项不起诉。
2. 根据前款规定不起诉或者撇销案件的，人民检察院、公安机关应当及时对查封、扣押、冻结的财物及其率息作出处理。
