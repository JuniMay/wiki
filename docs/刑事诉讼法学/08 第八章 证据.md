# 第八章 证据

## 证据的概念和意义

### 证据的概念

可以用于证明案件事实的材料，都是证据。

证据必须经过查证属实，才能作为 **定案的根据**。

### 证据的基本属性

- 传统证据三性说
    - 客观性：客观存在，不以人（犯罪分子、办案人员）的主观意志为转移
    - 关联性（相关性）：（区分于偶然性）
    - 合法性：收集过程要合法，表现形式要合法
- 现代证据的基本属性
    - 证明力
    - 证据能力（合法性）

### 证据的意义

1. 正确认定案件事实、适用法律的依据（唯一）
2. 惩罚犯罪、保护无辜的手段
3. 教育群众的重要形式

## 证据的收集原则

### 依法、全面、客观的原则

审判人员、检察人员、侦查人员必须依照法定程序，收集能够证实犯罪嫌疑人、被告人有罪或者无罪、犯罪情节轻重的各种证据。

必须保证一切与案件有关或者了解案情的公民，有客观充分地提供证据的条件。

除特殊情况外，可以吸收他们协助调查。

### 严禁刑讯逼供的原则

严禁刑讯逼供和以威胁、引诱、欺骗以及其他非法方法收集证据。

### 反对强迫自证其罪的原则

不得强迫任何人证实自己有罪

### 忠于事实真相的原则

公安机关提请批准逮捕书、人民检察院起诉书、人民法院判决书，必须忠实于事实真相。故意隐瞒事实真相的，应当追究责任。

凡是伪造证据、隐匿证据或者毁灭证据的，无论属于何方，必须受法律追究。

### 重证据、重调查研究、不轻信口供的原则

对一切案件的判处都要重证据，重调查研究，不轻信口供。

只有被告人供述，没有其他证据的，不能认定被告人有罪和处以刑罚；没有被告人供述，证据确实、充分的，可以认定被告人有罪和处以刑罚。

### 有关单位、个人配合的原则

1. 人民法院、人民检察院和公安机关有权向有关单位和个人收集、调取证据，有关单位和个人应当如实提供证据。
2. 行政机关在行政执法和查办案件过程中收集的物证、书证、视听资料、电子数据等证据材料，在刑事诉讼中可以作为证据使用

### 保密的原则

对涉及国家秘密、商业秘密、个人隐私的证据，应当保密。

## 证据的种类及其审查判断

### 物证

#### 概念

可以用于证明案件事实的一切物品和痕迹

#### 物证的审查判断

1. 审查物证的来源 → 应当有明确的出处
2. 审查物证与案件事实有无客观联系（证据的客观性、关联性）
3. 审查物证与其他证据之间有无矛盾

### 书证

#### 概念

以其文字、符号、图画等记载的内容和表达的思想来证明案件事实的一切书面文件。

#### 书证的审查判断

1. 审查书证是谁制作的
2. 审查书证是否被伪造
3. 审查书证是在何种情况下制作的
4. 审查书证的内容与案件事实有无联系
5. 审查书证与其他证据之间有无矛盾

### 证人证言

#### 概念

证人就其所了解的案件情况向公安司法人员所做的陈述

#### 证人的作证义务

凡是知道案件情况的人，都有作证的义务。

#### 证人的资格

生理上、精神上有缺陷或者年幼，不能辨别是非、不能正确表达的人，不能作证人。

#### 证人证言的审查判断

1. 审查证人证言的来源
2. 审查证人的品质、与当事人的关系、与案件的关系
3. 审查证人生理上、精神上有无缺陷。感知、记忆、表达能力如何
4. 审查证人证言的内容前后是否矛盾，与其他证据是否能够协调一致

#### 证人证言的质证、伪证的责任

证人证言必须在法庭上经过公诉人、被害人和被告人、辩护人双方质证并且查实以后，才能作为定案的根据。

法庭查明证人有意作伪证或者隐匿罪证的时候，应当依法处理。

#### 保护证人的原则性规定

人民法院、人民检察院和公安机关应当保障证人及其近亲属的安全。

对证人及其近亲属进行威胁、侮辱、殴打或者打击报复，构成犯罪的，依法追究刑事责任；尚不够刑事处罚的，依法给予治安管理处罚。

#### 特定案件中保护证人的规定

对于危害国家安全犯罪、恐怖活动犯罪、黑社会性质的组织犯罪、毒品犯罪等案件，证人、鉴定人、被害人因在诉讼中作证，本人或者其近亲属的人身安全面临危险的，人民法院、人民检察院和公安机关应当采取以下一项或者多项保护措施：

1. 不公开真实姓名、住址和工作单位等个人信息；
2. 采取不暴露外貌、真实声音等出庭作证措施；
3. 禁止特定的人员接触证人、鉴定人、被害人及其近亲属；
4. 对人身和住宅采取专门性保护措施；
5. 其他必要的保护措施。

#### 证人的保护请求权和有关单位、个人的配合义务

1. 证人、鉴定人、被害人认为因在诉讼中作证，本人或者其近亲属的人身安全面临危险的，可以向人民法院、人民检察院、公安机关请求予以保护。
2. 人民法院、人民检察院、公安机关依法采取保护措施，有关单位和个人应当配合

#### 证人的经济保障

1. 证人因履行作证义务而支出的交通、住宿、就餐等费用，应当给予补助。证人作证的补助列入司法机关业务经费，由同级政府财政予以保障
2. 有工作单位的证人作证，所在单位不得克扣或者变相克扣其工资、奖金及其他福利待遇。

### 被害人陈述

#### 概念

被害人就其侵害的事实和有关犯罪分子的情况向公安司法机关人员所做的陈述。

#### 被害人陈述的审查判断

1. 被害人与犯罪嫌疑人、被告人的关系
2. 被害人陈述的来源
3. 被害人陈述是否合情合理，前后有无锚段、与其他证据之间有无矛盾
4. 审查被害人思想品质和作风如何

### 犯罪嫌疑人、被告人供述和辩解

#### 概念

犯罪嫌疑人、被告人就有关案件情况向公安司法人员所做的陈述

#### 犯罪嫌疑人、被告人供述和辩解审查判断

1. 审查口供的动机
2. 审查口供的取得是否合法
3. 审查口供有无反复以及反复的原因
4. 审查口供的内容是否合情合理、前后有无矛盾、与其他证据之间有无矛盾

### 鉴定意见

#### 概念

公安司法机关为了解决案件中的专门性问题，指派或者聘请具有专门知识的人，进行鉴定后所作出的结论

#### 鉴定结论的审查判断

1. 鉴定人是否具有解决专门问题的知识和经验
2. 鉴定材料是否确实、充分
3. 鉴定设备是否完好、鉴定方法是否正确
4. 鉴定结论与其他证据之间是否存在矛盾
5. 鉴定人有无受到外界非法干扰

### 勘验、检查、辨认、侦查实验等笔录

#### 概念

公安司法人员对与案件有关的现场、物品、人身、尸体等进行勘验、检查、对犯罪嫌疑人的身份进行辨认或者为查明案件所进行的模拟实验所作出的笔录

#### 笔录的审查判断

1. 笔录的制作是否符合法律的要求
2. 笔录的内容是否符合客观实际
3. 对象有无遭到破坏或者伪造
4. 笔录记载的内容与案件事实有无联系，同其他证据之间有无矛盾

### 视听资料、电子数据

#### 概念

以录音、录像、电子计算机以及其他高科技设备储存的信息证明案件真实情况的证据

#### 视听资料的审查判断

1. 审查视听资料的形成过程。包括制作者、时间、地点、对象等
2. 审查视听资料是否被伪造、篡改
3. 审查视听资料的内容与案件事实有无关联，与其他证据之间是否能够协调一致

## 证据的分类

### 有罪证据、无罪证据

- 划分标准：证据的不同证明作用
- 凡是证明犯罪事实存在、犯罪嫌疑人实施犯罪行为的证据，都是有罪证据
- 凡是否定犯罪事实存在、证明犯罪嫌疑人、被告人没有实施犯罪行为的证据，都是无罪证据。
- 划分的意义

### 原始证据、传来证据

- 划分标准：证据的来源
- 凡是直接来源于案件事实的证据，都是原始证据
- 反不是直接来源于案件事实的证据，都是传来证据
- 划分的意义：证明力
- 传来证据与传闻证据（口头传闻、书面传闻、动作）的区别
    - 适用范围不一样：英美法系 → 实物证据不存在传闻，只有言词类；传来证据可以包括实物（不是原物）
    - 能否采用（证明力）
    - 与诉讼模式、司法理念有关（三段论 🆚 以审判为中心）

### 言词证据、实物证据

- 划分标准：证据的表现形式
- 凡是以言词作为表现形式的证据，都是言词证据
- 凡是以实体物的状态作为存在和表现形式的证据，都是实物证据
- 划分的意义
    - 收集方式方法不一样
    - 审查判断的方式不一样（与其它证据相结合）

### 直接证据、间接证据

- 划分标准：单独一个证据对案件主要事实的证明程度
- 案件的主要事实：犯罪事实是否发生，是否为犯罪嫌疑人、被告人所实施
- 凡是能单独证明案件主要事实的证据，都是主要证据
- 凡是不能单独证明案件的主要事实、而必须与其他证据结合起来，才能证明案件主要事实的证据，都是间接证据。（情况证据）
- 间接证据的作用 🔺
    - 是获得直接证据的先导
    - 是鉴别直接证据的有力手段
- 运用间接证据定案的规则
    - 每一个间接证据都必须查证属实
    - 每一个间接证据必须与案件有客观联系
    - 间接证据与间接证据之间、间接证据与案件事实之间必须协调一致，没有矛盾
    - 间接证据必须形成一个完整的证明体系（证据锁链），得出的结论具有排他性。

## 证明

### 证明对象

公安司法人员运用证据加以证明的案件事实

#### 有关犯罪的事实

1. 犯罪事实是否发生
2. 犯罪事实是否为犯罪嫌疑人、被告人实施
3. 实施犯罪行为的时间、地点、手段、情节
4. 犯罪行为与犯罪结果之间有无因果关系
5. 行为人是否具有刑事责任能力
6. 行为的目的、动机如何？故意还是过失
7. 行为是否属于正当防卫、紧急避险
8. 是否存在不追究刑事责任的情形
9. 其他与犯罪有关的事实

#### 有关犯罪嫌疑人、被告人的个人情况以及犯罪后的表现

e.g. 是否怀孕、是否未成年人、是否累犯，有无自首、立功表现，等

#### 有关程序方面的事实

e.g. 法官是否回避，是否非法证据，是否超过法定期限，等

### 举证责任（证明责任）

#### 概念

在案件事实真伪不明的情况下，由谁来承担败诉的责任

> 当作为裁判基础的法律要件事实在诉讼中处于真伪不明的状态时，一方当事人因此而承担的诉讼上的不利后果。

#### 我国刑事诉讼中举证责任的分担

1. 公诉案件中被告人有罪的举证责任由人民检察院承担；
2. 自诉案件中被告人有罪的举证责任由自诉人承担。

1. 被告人一般不承担举证责任[^1]
    1. 不需要自证无罪
    2. 不需要自证其罪

[^1]: [积极抗辩 - 维基百科，自由的百科全书](https://zh.m.wikipedia.org/zh-hans/%E7%A9%8D%E6%A5%B5%E6%8A%97%E8%BE%AF)

#### 有关举证责任的几个难点问题

1. 关于公安机关是否承担举证责任
2. 关于人民法院是否承担举证责任（应当认为不承担）
3. 举证责任的转移与倒置
4. 举证责任规范的意义：事实真伪不明情况下判案的方式
    1. 防止法官以事实不清为由拒绝裁判案件情况的发生
    2. 法官在事实真伪不明情况下裁判案件的依据

### 证明标准

#### 概念

承担证明责任的主体运用证据证明案件事实所要达到的程度

#### 如何理解“证据确实、充分”

1. 定罪量刑的事实都有证据证明；
2. 据以定案的证据均经法定程序查证属实；
3. 综合全案证据，对所认定事实已排除合理怀疑。

## 非法证据排除

### 非法证据的概念

采用刑讯逼供等非法方法收集的犯罪嫌疑人、被告人供述和采用暴力、威胁等非法方法收集的证人证言、被害人陈述，以及收集程序不符合法律规定的物证、书证。

### 绝对排除和相对排除

1. 采用刑讯逼供等非法方法收集的犯罪嫌疑人、被告人供述和采用暴力、威胁等非法方法收集的证人证言、被害人陈述，应当予以排除。
2. 收集物证、书证不符合法定程序，可能严重影响司法公正的，应当予以补正或者作出合理解释；不能补正或者作出合理解释的，对该证据应当予以排除。

### 排除非法证据的阶段

在侦查、审查起诉、审判时发现有应当排除的证据的，应当依法予以排除，不得作为起诉意见、起诉决定和判决的依据。

### 人民检察院在排除非法证据中的责任

人民检察院接到报案、控告、举报或者发现侦查人员以非法方法收集证据的，应当进行调查核实。对于确有以非法方法收集证据情形的，应当提出纠正意见；构成犯罪的，依法追究刑事责任。

### 人民法院在排除非法证据中的责任

1. 法庭审理过程中，审判人员认为可能存在本法第五十六条规定的以非法方法收集证据情形的，应当对证据收集的合法性
进行法庭调查。
2. 当事人及其辩护人、诉讼代理人有权申请人民法院对以非法方法收集的证据依法予以排除。

### 排除非法证据的先行举证责任及证明责任

1. 当事人及其辩护人、诉讼代理人有权申请人民法院对以非法方法收集的证据依法予以排除。申请排除以非法方法收集的证据的，应当提供相关线索或者材料。
2. 在对证据收集的合法性进行法庭调查的过程中，人民检察院应当对证据收集的合法性加以证明。

### 侦查人员的出庭说明情况义务

1. 现有证据材料不能证明证据收集的合法性的，人民检察院可以提请人民法院通知有关侦查人员或者其他人员出庭说明情况；
2. 人民法院可以通知有关侦查人员或者其他人员出庭说明情况。
3. 有关侦查人员或者其他人员也可以要求出庭说明情况。经人民法院通知，有关人员应当出庭。

### 非法证据的证明标准

对于经过法庭审理，确认或者不能排除存在本法第五十六条规定的以非法方法收集证据情形的，对有关证据应当予以排除。
