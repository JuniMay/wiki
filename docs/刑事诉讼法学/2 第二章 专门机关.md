# 第二章 专门机关


## 公安机关

- 性质：行政机关 治安保卫机关
- 组织：公安部、公安厅（局）、公安处（局）、公安局
- 在刑事诉讼中的职权—侦查、拘留、执行逮捕、预审
- 诉讼地位—主要侦查机关
- 其他侦查机关：
    - 国家安全机关 1983年
    - 军队保卫部门 1993年
    - 监狱 1994年
    - 海警局 2018年
    - 人民检察院
    - （监察委员会）

## 人民检察院

- 性质：司法机关    法律监督机关
- 组织：最高人民检察院、省（自治区、直辖市）人民检察院及其分院、市人民检察院、县（市）人民检察院
    
    检察委员会
    
- 在刑事诉讼中的职权—检察、批准逮捕、直接受理案件的侦查、提起公诉
- 在刑事诉讼中的地位
    - 侦查机关之一
    - 唯一的公诉机关
    - 法定的诉讼监督机关

## 人民法院

- 性质：司法机关
- 组织：最高人民法院、高级人民法院、中级人民法院、基层人民法院（包括派出法庭）
- 职权：审判
- 在刑事诉讼中的诉讼地位—唯一的审判机关
- 司法特征
    - 被动性、独立性
    - 中立性、公开性
    - 集中性、亲历性
    - 终结性（不可逆性）
- 审判组织
    - 独任制（基层人民法院适用简易程序、速裁程序的案件）
    - 合议制（基层、中级一审由审判员3人，或者由审判员和人民陪审员共三人或者七人）
    - 审判委员制
