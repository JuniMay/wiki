# 第七章 强制措施

## 强制措施的概念和意义

### 强制措施的概念

公检法在刑事诉讼活动中，为保证诉讼活动的顺利进行，依法对犯罪嫌疑人、被告人采取的限制或者剥夺其人身自由的各种强制方法

### 强制措施的意义

1. 有利于防止犯罪嫌疑人、被告人毁灭、伪造证据、串供
2. 有利于保障犯罪嫌疑人、被告人的人权不受非法侵犯（对程序有规定后，防止滥用）
3. 有利于防止犯罪嫌疑人、被告人继续犯罪

## 拘传

### 拘传的概念

指对于<mark>没有被拘留、逮捕</mark>的犯罪嫌疑人、被告人强制其到案接受讯问的一种强制措施

### 拘传的对象

1. 犯罪嫌疑人、被告人
2. 不需要逮捕、拘留
3. <mark>一般</mark>经合法传唤无正当理由拒不到案

### 拘传与传唤的区别

1. 法律性质不同：强制措施 🆚 通知
2. 适用对象不同：嫌疑人、被告人 🆚 既可以是嫌疑人、被告人，也可以是证人
3. 强制性不同

### 拘传的程序

#### 拘传的审批程序

公安局 → 局长 → 拘传证

人民检察院 → 检察长 → 拘传证

人民法院 → 院长 → 拘传票

#### 拘传的执行程序

1. 执行人员不得少于二人
2. 出示拘传证
3. 被拘传人签名、捺指印
4. 戒具的使用

### 执行拘传应当注意的三个问题

#### 拘传的期限

持续时间不超过12小时；案情特别重大、复杂，需要采取拘留、逮捕措施的，传唤、拘传持续时间不得超过<mark>24小时</mark>

#### 拘传的形式

不得以连续传唤、拘传的形式变相拘禁犯罪嫌疑人。传唤、拘传犯罪嫌疑人，应当保证犯罪嫌疑人的饮食和必要的休息时间

#### 拘传的地点

被拘传人所在市、县内的指定地点

## 取保候审

### 取保候审的概念

公检法依法责令犯罪嫌疑人、被告人提供担保，保证其不逃避侦查、起诉、审判，并随传随到的一种强制措施

### 取保候审的对象

1. 可能判处管制、拘役或者独立适用附加刑的
2. <mark>可能判处有期徒刑以上刑罚，采取取保候审不致发生社会危险性的</mark>
3. 患有严重疾病、生活不能自理，怀孕或者正在哺乳自己婴儿的妇女，采取取保候审不致发生社会危险性的
4. 羁押期限届满，案件尚未办结，需要采取取保候审的

### 取保候审的种类

#### 保证人担保

- 保证人的条件
   1. 与本案无牵连
   2. 有能力履行保证义务（对犯罪嫌疑人有威信）
   3. 享有政治权利，人身自由没有限制
   4. 有固定的住处和收入
- 保证人的义务
   1. 监督被保证人履行刑诉法 71 条的规定
   2. 发现被保证人可能发生或者已经发生违反本法 71 条规定的行为的，应当及时向执行机关报告
   3. 未履行保证义务的，对保证人处以罚款，构成犯罪的，承担刑事责任

#### 保证金担保

- <mark>不能与保证人担保合并使用</mark>

- 保证金的形式：人民币（现金）

- 保证金的数额：人民币 1000 元以上
  
    !!! note
  
        取保候审的决定机关应当综合考虑<mark>保证诉讼活动正常进行的需要，被取保候审人的社会危险性，案件的性质、情节，可能判处刑罚的轻重，被取保候审人的经济状况</mark>等情况，确定保证金的数额

- 保证金的收取与保管：提供保证金的人应当将保证金存入执行机关（公安机关）指定银行的专门账户

- 保证金的退还：犯罪嫌疑人、被告人在取保候审期间未违反本法第71条规定的，取保候审结束的时候，凭解除取保候审的通知或者有关法律文书到银行领取退还的保证金

### 取保候审的程序

#### 取保候审的审批 → 取保候审决定书

#### 取保候审的执行

- 取保候审的执行机关：公安机关
- 取保候审的执行程序
   1. 保证人担保的执行
   2. 保证金担保的执行 → 一次性交齐

#### 取保候审的期限 → 12 个月

### 被取保候审人应当遵守的义务

1. 未经执行机关批准不得离开所居住的市、县
2. 住址、工作单位和联系方式发生变动的，在二十四小时内向执行机关报告
3. 在传讯的时候即使到岸
4. 不得以任何形式干扰证人作证
5. 不得毁灭、伪造证据或者串供

人民法院、人民检察院和公安机关可以根据案件情况，责令被取保候审的犯罪嫌疑人、被告人遵守以下一项或者多项

1. 不得进入特定的场所;
2. 不得与特定的人员会见或者通信
3. 不得从事特定的活动;
4. 将护照等出入境证件、驾驶证件交执行机关保存。

### 被取保候审人的责任

!!! note

    　　被取保候审的犯罪嫌疑人、被告人违反前两款规定，已缴纳保证金的，没收部分或者全部保证金，并且区别情形，责令犯罪嫌疑人、被告人具结悔过，重新交纳保证金、提出保证人，或者监视居住、予以逮捕。
    　　对违反取保候审规定，需要予以逮捕的，可以对犯罪嫌疑人、被告人先行拘留。

### 取保候审的解除（与监视居住同）

!!! note

    　　在取保候审期间，<mark>不得中断</mark>对案件的侦查、起诉和审查。对于发现不应当追究刑事责任或者取保候审期限届满的，应当<mark>及时解除</mark>取保候审。解除取保候审，应当<mark>及时通知</mark>被取保候审人和有关单位

## 监视居住

### 监视居住的概念

公检法依法责令犯罪嫌疑人、被告人不得擅自离开<mark>住处</mark>[^1]或者指定的居所，并对其行为加以监视的一种强制措施

[^1]: 可以理解为小区。

### 监视居住的对象

- <mark>符合逮捕条件</mark>
- 有下列情形之一：
   1. 患有严重疾病、生活不能自理的
   2. 怀孕或者正在哺乳自己婴儿的妇女
   3. 系生活不能自理的人的唯一扶养人
   4. 因为案件的特殊情况或者办理案件的需要，采取监视居住措施更为适宜的
   5. 羁押期限届满，案件尚未办结，需要采取监视居住措施的
   6. 符合取保候审条件，但犯罪嫌疑人、被告人不能提出保证人，也不交纳保证金的

### 监视居住的执行机关 → 公安机关

### 监视居住的期限 → 六个月

### 监视居住的执行地点

1. 监视居住应当在犯罪嫌疑人、被告人的<mark>住处</mark>执行
2. 无固定住处的，可以在<mark>指定居所</mark>执行
3. 对于涉嫌危害国家安全犯罪、恐怖活动犯罪，在住处执行<mark>可能有碍侦查</mark>的，经上一级人民检察院或者公安机关批准，也可以在<mark>指定的居所</mark>执行。但是，<mark>不得在羁押场所、专门的办案场所执行</mark>。

### 通知家属

### 委托辩护人的权利及相关规定

1. 犯罪嫌疑人、被告人被监视居住的，也可由其监护人、近亲属代为委托辩护人（被监视居住的犯罪嫌疑人、被告人委托辩护人，适用本法第三十四条的规定）

2. 辩护律师可以同被监视居住的犯罪嫌疑人、被告人会见和通信。其他辩护人经人民法院、人民检察院许可，也可以同被监视居住的犯罪嫌疑人、被告人会见和通信

3. 危害国家安全犯罪、恐怖活动犯罪案件，在侦查期间辩护律师会见被监视居住的犯罪嫌疑人，应当经侦查机关许可
   
    上述案件，侦查机关应当事先通知看守所

4. 辩护律师对被监视居住的犯罪嫌疑人、被告人，可以了解案件有关情况，提供法律咨询等；自案件移送审查起诉之日起，可以向犯罪嫌疑人、被告人核实有关证据。辩护律师会见犯罪嫌疑人、被告人时不被监听

### 被监视居住人应当遵守以下规定

1. 未经执行机关批准不得离开执行监视居住的住所
2. 未经执行机关批准不得会见他人或者通信
3. 在传讯的时候即使到案
4. 不得以任何形式干扰证人作证
5. 不得毁灭、伪造证据或者串供
6. 将护照等出入境证件、身份证件、驾驶证件交执行机关保存。

被监视居住的犯罪嫌疑人、被告人违反前款规定，情节严重的，可以予以逮捕;需要予以逮捕的，可以对犯罪嫌疑人、被告人先行拘留。

### 监视方法

!!! note

    执行机关对被监视居住的犯罪嫌疑人、被告人，可以采取电子监控、<mark>不定期检查</mark>等监视方法对其遵守监视居住规定的情况进行监督;在侦查期间，可以对被监视居住的犯罪嫌疑人的<mark>通信进行监控</mark>。

### 监视居住的解除（与取保候审同）

!!! note

    在监视居住期间，不得中断对案件的侦查、起诉和审理。对于发现不应当追究刑事责任或者监视居住期限届满的，应当及时解除监视居住。解除监视居住，应当及时通知被监视居住人和有关单位。

### 刑期折抵

!!! note

    指定居所监视居住的期限应当折抵刑期。被判处管制的，监视居住一日折抵刑期一日；被判处拘役、有期徒刑的，监视居住二日折抵刑期一日。

### 检察监督

!!! note

    人民检察院对指定居所监视居住的决定和执行是否合法实行监督。

## 逮捕

### 逮捕的概念

公检法依法采取的剥夺犯罪嫌疑人、被告人的人身自由，并且将其羁押的一种强制措施。

### <mark>逮捕的条件</mark>

1. <mark>**有证据证明有犯罪事实**</mark>
    1. 有证据证明发生了犯罪事实; （立案条件）
    2. 有证据证明犯罪事实是犯罪嫌疑人、被告人实施的;
    3. 证明犯罪嫌疑人实施犯罪行为的证据<mark>已有查证属实的</mark>。
2. <mark>可能判处徒刑以上刑罚，</mark>
3. <mark>采取取保候审尚不足以防止发生 **社会危险性**。</mark>
    1. 可能实施新的犯罪的;
    2. 有危害国家安全、公共安全或者社会秩序的现实危险的;
    3. 可能毁灭、伪造证据，干扰证人作证或者串供的;
    4. 可能对被害人、举报人、控告人实施打击报复的;
    5. 企图自杀或者逃跑的。
    6. 可能判处十年有期徒刑以上刑罚的
    7. 曾经故意犯罪或者身份不明的
    8. 违反取保候审、监视居住规定，情节严重的
    9. 拒不认罪认罚

### 逮捕的程序

!!! note
    逮捕犯罪嫌疑人、被告人，必须经过人民检察院批准（公安提交）<mark>或者决定</mark>（自侦案件），或者人民法院决定（审理过程中），由公安机关执行

#### 公安机关逮捕犯罪嫌疑人的程序

1. 公安机关提请
   
    公安机关要求逮捕犯罪嫌疑人的时候，应当写出提请批准逮捕书，连同案卷材料、证据，一并移送同级人民检察院审查批准。

2. 人民检察院审查
   
    - 人民检察院审查批准逮捕，<mark>可以讯问</mark>犯罪嫌疑人;有下列情形之一的，<mark>应当讯问<mark>犯罪嫌疑人:
       - 对是否符合逮捕条件有疑问的;
       - 犯罪嫌疑人要求向检察人员当面陈述的;
       - 侦查活动可能有重大违法行为的
    - 人民检察院审查批准逮捕，可以<mark>询问</mark>证人等诉讼参与人，听取辩护律师的意见；辩护律师提出要求的，<mark>应当</mark>听取辩护律师的意见。
    - 必要的时候，人民检察院可以派人参加公安机关对于重大案件的讨论

3. 人民检察院审查后的决定
   
    人民检察院审查批准逮捕犯罪嫌疑人由检察长决定。重大案件应当提交检察委员会讨论决定。
   
    - 批准逮捕；
    - 不批准逮捕(需要补充侦查的，应当通知公安机关)

4. 向人民检察院通知
   
    - 对于批准逮捕的决定，公安机关应当立即执行，并且将执行情况及时通知人民检察院。
    - 对于不批准逮捕的，人民检察院应当说明理由，需要补充侦查的，应当同时通知公安机关。

5. 公安机关的复议、复核申请
   
    !!! note
   
        公安机关对人民检察院不批准逮捕的决定，认为有错误的时候，可以要求复议，但是必须<mark>将被拘留的人立即释放</mark>。如果意见不被接受，可以向上一级人民检察院提请复核。上级人民检察院应当立即复核，作出是否变更的决定，通知下级人民检察院和公安机关执行。

#### 人民检察院逮捕犯罪嫌疑人的程序

人民检察院决定、交给公安机关执行

#### 人民法院逮捕被告人的程序

人民法院决定，交给公安机关执行

### 逮捕的执行

1. 逮捕由公安机关执行

2. 逮捕证的签发（公安签发）

3. 逮捕证的出示、被逮捕人的签名

4. 异地逮捕的执行
   
    !!! note
   
        公安机关在异地执行拘留、逮捕的时候，应当通知被拘留、逮捕人所在地的公安机关，被拘留、逮捕人所在地的公安机关应当予以配合

5. 羁押场所、逮捕后对家属的通知
   
    !!! note
   
        逮捕后，应当<mark>立即</mark>将被逮捕人<mark>送看守所</mark>羁押。除无法通知的以外，应当在逮捕后<mark>二十四小时</mark>以内，通知被逮捕人的家属

6. 逮捕后的讯问
   
    !!! note
   
        人民法院、人民检察院对于各自决定逮捕的人，公安机关对于经人民检察院批准逮捕的人，都必须在逮捕后的<mark>二十四小时以内进行讯问</mark>。在发现不应当逮捕的时候，必须立即释放，发给释放证明。

7. 检察监督
   
    !!! note
   
        犯罪嫌疑人、被告人被逮捕后，人民检察院仍应当对<mark>羁押的必要性</mark>进行审查。对不需要继续羁押的，应当建议予以释放或者变更强制措施。有关机关应当在<mark>十日以内</mark>将处理情况通知人民检察院。

## 拘留

### 拘留的概念

!!! note

    公安机关、人民检察院对于现行犯（正在进行）或者重大嫌疑分子，依法采取的<mark>临时</mark>（和逮捕相比）剥夺其人身自由的一种强制措施（主要公安机关，人民检察院也有，人民法院没有）

### 拘留的情形

1. 正在预备犯罪、实行犯罪或者在犯罪后即时被发觉的;
2. 被害人或者在场亲眼看见的人指认他犯罪的;
3. 在身边或者住处发现有犯罪证据的;
4. 犯罪后企图自杀、逃跑或者在逃的;
5. 有毁灭、伪造证据或者串供可能的;
6. 不讲真实姓名、住址，身份不明的;
7. 有流窜作案、多次作案、结伙作案重大嫌疑的。
   对于以上第（四）、（五）项的情形，人民检察院可以实施拘留。

### 拘留的程序

1. 拘留的审批 → 签发拘留证

2. 拘留的执行 → 与逮捕相同

3. 拘留后的告知和讯问 → 与逮捕相同

4. 羁押场所、通知
   
    !!! note
   
        拘留后，应当立即将被拘留人送<mark>看守所</mark>羁押，<mark>至迟不得超过二十四小时</mark>。
   
    !!! note
   
        除无法通知或者涉嫌危害国家安全犯罪、恐怖活动犯罪通知可能有碍侦查的情形以外，应当在拘留后二十四小时以内，通知被拘留人的家属。有碍侦查的情形消失以后，应当立即通知被拘留人的家属

### 公安机关拘留人的期限

|               | 公安提请期限 | 检察院审批期限 | 总期限    |
| ------------- | ------ | ------- | ------ |
| 一般情况          | 3日     | 7日      | 10日    |
| 特殊情况          | 4-7日   | 7日      | 11-14日 |
| 流窜作案（跨地区连续作案） |        |         |        |
| 多次作案（三次及三次以上） |        |         |        |
| 结伙作案（两人及两人以上） | 30日    | 7日      | 37日    |

### 人民检察院拘留嫌疑人的期限

!!! note

    人民检察院对直接受理的案件中被拘留的人，认为需要逮捕的，应当在十四日以内作出决定。

!!! note

    在特殊情况下，决定逮捕的时间可以延长一日至三日。

### 扭送（十分紧急）

1. 正在实行犯罪或者犯罪后即时被发觉的
2. 通缉在案的
3. 越狱逃跑的
4. 正在被拘捕的

## 强制措施的变更和解除

### 依职权撤销或者变更强制措施

!!! note

    人民法院、人民检察院和公安机关如果发现对犯罪嫌疑人、被告人采取强制措施不当的，应当及时撤销或者变更。

!!! note

    公安机关释放被逮捕的人或者变更逮捕措施的，应当通知原批准的人民检察院

### 申请变更强制措施

!!! note

    犯罪嫌疑人、被告人及其法定代理人近亲属或者辩护人有权申请变更强制措施。

!!! note

    人民法院、人民检察院和公安机关收到申请后，应当在三日以内作出决定;不同意变更强制措施的，应当告知申请人，并说明不同意的理由。

### 羁押超期的释放及变更

!!! note

    犯罪嫌疑人、被告人被羁押的案件，不能在本法规定的侦查羁押、审查起诉、一审、二审期限内办结的，对犯罪嫌疑人、被告人应当予以释放;需要继续查证、审理的，对犯罪嫌疑人、被告人可以取保候审或者监视居住。

### 强制措施期满的解除

!!! note

    人民法院、人民检察院或者公安机关对被采取强制措施法定期限届满的犯罪嫌疑人、被告人，应当予以释放、解除取保候审、监视居住或者依法变更强制措施。

!!! note

    犯罪嫌疑人、被告人及其法定代理人、近亲属或者辩护人对于人民法院、人民检察院或者公安机关采取强制措施法定期限届满的，有权要求解除强制措施。

### 人民检察院对强制措施的监督

!!! note

    人民检察院在审查批准逮捕工作中，如果发现公安机关的侦查活动有违法情况，应当通知公安机关予以纠正公安机关应当将纠正情况通知人民检察院。
